Конвертер архивов из формата полигона в формат бакса.

Как использовать конвертер:

Сначала надо его собрать. Для этого есть два способа.

Способ первый. Простой:
Заходите на https://github.com/PloadyFree/polybacs/releases и скачиваете оттуда jar-файл.

Способ второй. Посложнее
Скачиваете весь проект.
Затем из консоли зайти в папку с проектом и собрать его командой "./mvnw package" (без кавычек).
Затем взять получившийся jar-ник из target/musin.polybacs-1.0-SNAPSHOT.jar и (желательно) переименовать его во что-то попроще. Например, в "qq.jar".

Далее об использовании:
Берём этот jar-ник и кладём в какую-нибудь папку, рядом с ним кладём нужные zip-архивы полигона.
В консоли переходим в ту папку, куда положили jar-ник, и набираем "java -jar qq.jar" (без кавычек).
В случае успеха появится папка ready с готовыми архивами.

Если в какой-то из моментов консоль сказала, что она не понимает, что такое java или javac, значит в PATH не прописан путь до jdk.
Пропишите его (вплоть до папки bin, то есть, например, C:\Program Files\Java\jdk1.8.0_192\bin) и перезапустите консоль.
Теперь всё должно заработать.
